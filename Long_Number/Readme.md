# LongNumber — длинная арифметика на C++

## О проекте

Библиотека для работы с целыми числами произвольной длины.  
Реализована с нуля, без использования сторонних библиотек (кроме Google Test).

## Цель проекта:
- Управления памятью в C++
- Алгоритмов длинной арифметики
- Модульного тестирования
- Структуры проекта и сборки (CMake)

## Возможности

- **Арифметика**: +, -, *, /, %, унарный минус
- **Сравнение**: ==, !=, >, <
- **Присваивание**: копирование, перемещение, из строки
- **Конструкторы**: по умолчанию, из строки, копирования, перемещения
- **Работа со знаком**: отрицательные числа поддерживаются
- **Нормализация**: все нули (включая -0) приводятся к положительному нулю
- **Тесты**: более 40 тестов Google Test

## Работа с проектом:
   - Использование(только Linux):
    chmod +x build_module_app.sh
    ./build_module_app.sh

   - Запуск проекта(только Linux):
    cd build_ninja/app
    ./menu_app

   - Запуск тестирования(только Linux):
    cd build_ninja/test
    ./test_LN

## Технологии:
   - ЯП: C++
   - Система сборки: CMake
   - Сборщик: Ninja
   - Тестирование: Google Test
