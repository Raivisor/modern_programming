проект на C++ с модульной архитектурой, автоматической сборкой через CMake и юнит-тестами.

Цели проекта:
1. Обучение: сборка проекта при помощи Cmake и Unit тестирования
2. Практическая: построение архитектуры проекта
3. Техническая: автоматизация сборки проекта и тестирования модулей

Использование(только Linux):
chmod +x build_module_app.sh
./build_module_app.sh

Запуск проекта menu(только Linux):
cd build_ninja/menu/app
./menu_app

Запуск проекта sort(только Linux):
cd build_ninja/sort/app
./sort_app

Запуск тестирования сортировки(только Linux):
cd build_ninja/sort/test
./test_shell

Технологии:
ЯП - C++
Система сборки - Cmake
Сборщик - Ninja
Тестирование - Google Test
